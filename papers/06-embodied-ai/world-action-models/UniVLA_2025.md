## 一句话总结

UniVLA 用一个 8.5B 纯自回归 Transformer（架构沿用 Emu3）把视觉、语言、动作全部离散化为同一词表内的 token 并交织建模，先用无动作标注的大规模机器人视频做"世界模型"式后训练捕捉时序因果动态,再用少量动作数据微调,在 LIBERO 上取得 95.5% 的平均成功率（超过 π0-FAST 的 85.5%）,在 CALVIN、SimplerEnv-Bridge 上同样刷新 SOTA,并进一步验证了该统一 token 范式向自动驾驶（NAVSIM）的迁移能力。

## 一、问题与动机

现有 VLA 模型大多遵循"语言中心"的后融合范式：先用预训练 VLM（如 PaliGemma、Qwen2-VL）把图像投影到语义空间,再在此表征之上派生动作策略（如 RT-2、OpenVLA、π0）。作者指出这种范式有两个结构性缺陷：

1. **模态异构但被强行对齐**：视觉是高维连续的空间信号,语言是抽象离散的语义符号,动作是具有时序因果依赖的连续序列——后融合只学到"表征对齐",难以形成真正紧耦合的跨模态表示。
2. **感知-动作回路本质是动态因果的,但现有 VLA 是静态映射**：多数方法只学"单帧图像 → 动作"的映射,丢弃了视频中丰富的时序信息,无法很好地利用大规模无动作标注的视频数据做自监督学习。

由此提出核心问题：能否在一个统一的离散 token 空间中联合建模视觉、语言、动作,以获得更紧密的跨模态整合和更有效的策略学习？

## 二、核心方法

### 2.1 统一多模态表示

UniVLA 放弃了 ViT 编码器 + 独立动作头的常规架构,改为**无编码器（encoder-free）**设计：三种模态各自离散化后拼接进同一条序列 $L$。

- **语言 token** $L_t$：标准文本 tokenizer。
- **视觉 token** $L_v$：沿用 Emu3 的 VQ 图像 tokenizer,空间压缩因子为 8,把每帧图像离散化为一组视觉 codebook token。
- **动作 token** $L_a$：沿用 FAST 方案,先对动作序列做 1/99 百分位归一化,再用离散余弦变换（DCT）把连续动作序列 $A_{1:H}=\{a_1,\dots,a_H\}$（每个 $a_t$ 为 $d$ 维向量,窗口大小 $H$）编码为一段变长的离散 token 序列 $[T_1,\dots,T_n]$,词表大小 1024,直接复用/替换语言 tokenizer 词表的末尾 1024 个 ID,从而与文本、视觉共享同一词表空间。
- 特殊边界符 `<boi>/<eoi>`（图像起止）与 `<boa>/<eoa>`（动作起止）用于在序列中划定不同模态片段。

训练目标统一简化为标准的 next-token 预测交叉熵损失,通过对不同任务只在特定 token 位置计算 loss 来适配多任务格式（纯语言监督做感知定位、纯视觉监督做世界建模、纯动作监督做策略学习）。

### 2.2 用 MDP 视角统一序列建模

作者把具身规划形式化为马尔可夫决策过程（MDP）：指令与当前观测决定动作,动作改变环境产生新观测,新观测再指导下一步动作。基于这一交织的马尔可夫结构,论文给出两类任务特定的序列模板：

**世界模型（后训练阶段）**：把语言指令视为一种广义的"动作",给定当前观测 $L_v^1$ 与指令 $L_t^1$,模型需要预测未来的视觉内容,loss 只施加在视觉 token 上：

$$S_v = \{L_t^1,\ L_v^1,\ L_v^2,\ \dots,\ L_v^t\}$$

用大白话说：这一步完全不需要动作标注,只靠"看指令 + 看当前帧,猜接下来几帧长什么样"来学习环境的时序因果动态（近似传统世界模型中的状态转移 $P(\mathbf{s}_{t+1}\mid \mathbf{s}_t,\mathbf{a}_t)$,只不过这里把指令当作动作的代理）,因此可以直接吃海量无动作标注的机器人视频。

**策略学习（微调阶段）**：loss 只施加在动作 token 上,序列显式交织视觉观测与动作：

$$S_a = \{L_t^1,\ L_v^1,\ L_a^1,\ L_v^2,\ L_a^2,\ \dots,\ L_v^t,\ L_a^t\}$$

用大白话说：这一步才真正学"看到什么就该做什么",但由于前一阶段已经把"世界怎么变化"学得比较好,微调阶段能更快、更省数据地收敛到好的策略。

两阶段合起来构成完整训练流程：(1) 用 VL 对齐的 Emu3 权重初始化,做无动作的世界模型后训练；(2) 在此基础上做交织动作的策略微调。论文额外做了消融（见下）验证：即便没有专门的后训练阶段,微调时只要保留视觉预测 loss,也能借自回归特性"顺带"学到一部分世界模型效果。

## 三、实验结果

模型规模：8.5B 参数,架构与 Emu3 一致。后训练数据集：从 RT-1、BridgeV2、DROID、Kuka、TOTO、Taco Play、FMB、Berkeley Autolab UR5、VIOLA、CMU Play Fusion、UT Austin Mutex（真实）以及 CALVIN、LIBERO、ManiSkill2（仿真）等数据集筛选,共 622K 段视频（Kuka 数据量过大,随机降采样保留 10 万段）,batch size 64 训练 30K 步,仅在视觉 token 上计算 loss。

关键发现：**纯动作空间的后训练反而是负收益**（各基准全线下降）,原因是跨数据集动作空间不一致、迁移性差；而任何形式的视觉监督后训练都显著正收益,其中"指令 + 视频"联合的世界模型目标收益最大。

**数据/训练效率（Table 5）**：CALVIN 上仅用 10% 微调数据,UniVLA 达到 Avg Len 3.19,远超 RoboVLMs（2.52）、GR-1（2.00）,且无后训练版本仅 0.15,说明收益主要来自世界模型后训练而非架构本身。SimplerEnv-Bridge 上 UniVLA 用 12k 步、batch 128 收敛到 64.6% 成功率,而 RoboVLMs 需要 50k 步才到 37.5%。

**跨域迁移：NAVSIM 自动驾驶（Table 7,PDMS 综合分）**：UniVLA 仅用前视相机（FC）、无 BEV/多传感器融合、未经过驾驶视频预训练,微调后 PDMS 达 81.7,接近使用 BEV+多相机+激光雷达的 Transfuser（84.0）、UniAD（83.4）、VADv2（83.0）,验证统一自回归 token 范式的跨任务通用性。

## 四、局限性

- 论文自陈"由于计算资源有限,对后训练可扩展性（scaling）的研究仍处于早期阶段",目前 622K 视频规模下已见明显收益,但尚未验证更大规模视频数据下的 scaling 曲线。
- 统一多模态框架尚未与强化学习范式结合,论文明确指出这是未来工作方向,当前完全依赖模仿学习式的下一 token 预测。
- 纯动作后训练的负迁移现象（Table 4）表明该框架对训练任务/数据配比较敏感——如果后训练阶段引入的视觉监督比例或数据来源不当,容易适得其反,这对后续复现和扩展提出了配比调参的隐性成本。
- 自动驾驶实验只是"预初步验证潜力"（preliminary transfer）,模型未在驾驶视频上做过世界模型式后训练,只是直接微调,因此并不能说明该统一范式在自动驾驶上已具备与专用 BEV 方法同等的成熟度。
- 离散化本身（VQ 视觉 token + DCT 动作 token）会引入量化误差,论文未讨论该误差对高精度接触式操作（如插拔、装配）的影响上限。

## 五、评价与展望

**优点**：UniVLA 的核心贡献是把"视觉-语言-动作统一 token 化 + 自回归建模"这一想法落到了具体、可复现的工程方案上,并且用一组干净的消融（Table 4/5/6）把"世界模型后训练为什么有效"拆解得比较清楚——尤其是"纯动作后训练是负收益,纯视觉/图文后训练才是正收益"这一发现,对后续 VLA 预训练配方的设计具有参考价值：它从实证角度提示,动作标签的跨数据集异构性可能是阻碍大规模预训练的瓶颈,而与动作解耦的视觉时序信号才是更适合大规模自监督学习的信号源。跨到自动驾驶（NAVSIM）的迁移实验也从侧面印证了该 token 化范式并非针对操作任务的窄适配。

**与同类公开工作的关系**：UniVLA 与 GR-1/GR-2、LAPA、AdaWorld、UVA 等"视频预训练 + 策略学习"路线同属一个大方向,但选择了与 latent-action 方法（LAPA、Moto、AdaWorld）不同的路径——不引入额外的隐动作 codebook,而是直接复用同一套离散视觉 token 做"世界模型式"视觉预测监督,声称在迁移性上优于隐动作范式（论文原文提到"our world model offers a simpler paradigm without latent actions, achieving better transferability",但未给出与 LAPA/AdaWorld 在同一基准上的直接数值对比,这一优势论断目前主要是定性表述）。相较于 π0/π0-FAST 的流匹配或频域离散动作头,UniVLA 更彻底地把动作也纳入统一自回归 token 空间,牺牲了连续动作头可能带来的精细控制精度,换取了架构统一性和多任务（感知定位/视觉预测/策略学习）共享同一模型的灵活性。

**开放问题与可能方向**：(1) 后训练视频规模和多样性的 scaling law 尚未被充分探索,622K 段视频相对于 Emu3/世界模型社区的数据规模仍偏小；(2) 纯动作负迁移现象背后更细致的机制（是动作空间尺度不一致,还是不同机器人本体的动力学差异）值得进一步分离验证；(3) 论文未系统对比 UniVLA 与 latent-action 类方法在相同数据、相同基准下的直接数字,后续工作可以补齐这一空缺以验证"无隐动作更具迁移性"的论断；(4) 离散化动作 token 在高精度接触式任务上的精度上限,以及与连续动作头（如扩散/流匹配头）的混合方案,是值得跟进的工程问题；(5) 论文提出的"和 RL 范式结合"仍是空白,如何在离散 token 自回归框架下做在线策略优化或离线 RL 微调是自然的后续方向。

## 参考

- Emu3（Wang et al., 2024）——UniVLA 的图像 tokenizer 与基座架构来源。
- FAST（Pertsch et al., 2025）——DCT 离散动作 tokenizer,UniVLA 动作离散化方案的直接来源。
- π0 / π0-FAST（Black et al., 2024/2025）——流匹配 VLA 与频域动作离散化基线,LIBERO/CALVIN 主要对比对象之一。
- OpenVLA（Kim et al., 2024）与 RoboVLMs（Li et al., 2024）——语言中心后融合 VLA 代表性基线。
- LAPA / AdaWorld（Ye et al., 2025; Gao et al., 2025）——同属"视频/世界模型驱动策略学习"方向的隐动作（latent-action）路线,与本文的直接 token 化路线构成方法论对照。
