## 一句话总结

提出 3D-centric 的 World-Spatial-Action（WSA）建模范式，用 Mixture-of-Transformers 的三专家结构（2D 空间专家、3D 空间专家、3D 动作专家）加双向因果注意力，把"3D 世界演化预测""3D 一致的 2D 视觉思考""3D 逆动力学"耦合进同一隐空间联合训练；仅用 6000 小时异构数据预训练（其中真机仅约 1000 小时），WSA₁ 在 RoboTwin2.0（hard 设置）取得 93.1% 成功率，为开源模型中最优，并在 7 项真机双臂/单臂任务上相对第二名 π0.5 提升约 +20 个百分点（77.5%/80.3% vs 54.9%）。

## 一、问题与动机

Robot Foundation Models（RFM）目前收敛为两条路线：

- **VLA**（如 RT-2、OpenVLA、π0/π0.5）："理解后执行"，把动作专家接到预训练 VLM 后面直接从 2D 观测预测动作，本质是 2D 语义到动作的单向映射；
- **WAM**（World Action Model，如 Motus、Fast-WAM、GeoPredict）："想象后执行"，先用视频/世界生成模型预测未来帧或未来 3D 场景作为视觉/空间子目标，再用逆动力学模型把预测的视觉/空间状态转成动作。

论文指出两个根本问题：

1. 无论 VLA 还是既有 WAM（含 3D WAM 如 GeoPredict），都缺乏对 3D 世界动力学和物理动作因果的显式互相约束建模——多是单向的"预测然后执行"，动作不会反过来影响世界预测，3D 场景状态和动作生成也没有互为条件；
2. 模仿学习高度依赖大规模真机示范数据，采集成本高昂、难以规模化，数据稀缺进一步限制了模型学习可泛化的物理常识。

核心问题："如何构造一个数据高效的建模范式，让机器人从示范中联合建模 3D 世界演化与物理行为之间的相互依赖，从而突破朴素模仿学习的泛化瓶颈？"

## 二、核心方法

### 2.1 四种建模范式的统一记号（对应 Figure 2）

令 $O_t=\{v_t,s_t,l\}$ 为当前视觉观测/本体感知/语言指令，$A_t$ 为动作块，$G_t$ 为 3D 场景表征序列，四类策略可写作：

- 2D VLA：$\mathbf{A}_t \sim p(a_{t+1:t+H}\mid v_t,s_t,l)$ —— 直接从 2D 观测预测动作；
- 2D WAM：$\mathbf{A}_t \sim p(a_{t+1:t+H}, v_{t+1:t+N}\mid v_t,s_t,l)$ —— 联合预测 2D 子目标图像与动作；
- 3D WAM（如 GeoPredict）：$\mathbf{A}_t \sim p(a_{t+1:t+H}, g_{t+1:t+K}\mid g_t,v_t,s_t,l)$ —— 单向地先预测 3D 场景演化再生成动作；
- 本文 3D WSA：不是单向映射，而是联合建模三个互相施加一致性约束的条件分布：

$$p(g_{t+1:t+K} \mid a_{t+1:t+H}, O_t), \quad p(v_{t+1:t+N} \mid g_{t+1:t+K}, O_t), \quad p(a_{t+1:t+H} \mid g_{t+1:t+K}, O_t)$$

用大白话说：第一项是"动作会怎样改变 3D 世界"（世界因果层），第二项是"给定 3D 世界目标，2D 画面子目标长什么样"（空间认知层，把 2D 视觉想象锚定到 3D 几何上），第三项是"给定 3D 世界状态变化，应该执行什么动作"（动作执行层，即 3D 逆动力学）。三者共享同一隐空间联合训练，形成"动作→3D 世界预测→2D 视觉子目标→3D 逆动力学生成动作"的闭环。

### 2.2 模型架构：三专家 Mixture-of-Transformers

WSA₁ 基于 Mixture-of-Transformers（MoT），包含三个互补专家：

- **2D 空间专家（2D-SE）** $f_{2D}(\cdot)$：由预训练 VLM（Qwen3-VL 或 Wan2.2）初始化，预测的不是 RGB 像素而是高度抽象的视觉 token $h_v=f_{2D}(O_t,G_t)$ 来表示子目标图像，聚焦图像本质特征而非冗余像素信息；
- **3D 空间专家（3D-SE）** $f_{3D}(\cdot)$：与 2D-SE 同深度的 Transformer，预测 3D 场景隐表征 $h_g=f_{3D}(O_t,A_t)$，即给定动作序列预测未来 3D 场景状态（对应认知科学中"运动的前向内部模型"）；
- **3D 动作专家（3D-AE）** $f_{act}(\cdot)$：基于 flow matching 的去噪扩散 Transformer，输入带噪动作 $\hat{A}_t=\tau A_t+(1-\tau)\omega$（$\omega\sim\mathcal{N}(0,1)$，$\tau\in[0,1]$），通过迭代交叉注意力和去噪预测隐动作表征 $h_{act}=f_{act}(O_t,G_t,\hat{A}_t)$，本质是把 3D 场景演化映回可执行动作的逆动力学。

### 2.3 3D 中心的世界-动作因果注意力

三专家通过共享注意力机制耦合，遵循以下依赖规则：

1. 所有预测 token（$h_v,h_g,h_{act}$）都可以 attend 到当前观测 token，确保一切预测锚定在机器人和环境的当前状态；
2. 未来子目标图像的视觉 token 可以 attend 到 3D 场景 token $h_g$，学习 2D-3D 视觉一致性，让语义视觉预测扎根于物理一致的 3D 几何；
3. **3D 场景 token $h_g$ 和动作 token $h_{act}$ 之间是完全双向注意力**——这是范式的关键：动作生成不仅参考 3D 场景预测，3D 场景预测也反过来受动作生成约束，二者互为条件，而非单向的相关性映射。

### 2.4 三个训练目标

- **3D 世界感知视觉思考（3D WA-VT）**：对预测视觉 token 做 MSE，与用预训练 tokenizer（如 COSMOS）编码真实子目标图像 $V_t$ 得到的 ground-truth token 对齐：

$$\mathcal{L}_{2D} = \mathbb{E}_{(O_t,G_t,V_t)\sim\mathcal{D}_{\text{dem}}}\left\|f_{2D}(O_t,G_t)-f_{enc}(V_t)\right\|_2^2$$

- **动作条件 3D 世界建模（3D AC-WM）**：对预测 3D 场景隐表征做 MSE，ground truth 由预训练 3D 基础模型（Depth-Anything）提取：

$$\mathcal{L}_{3D} = \mathbb{E}_{(O_t,A_t,G_t)\sim\mathcal{D}_{\text{dem}}}\left\|f_{3D}(O_t,A_t)-f_g(G_t)\right\|_2^2$$

- **3D 逆动力学建模（3D IDM）**：flow matching 损失，最小化预测流场与真实流场 $(\omega-A_t)$ 之差：

$$\mathcal{L}_{\text{ACT}} = \mathbb{E}_{(O_t,G_t,A_t)\sim\mathcal{D}_{\text{dem}}}\left\|\omega-A_t-f_{act}(O_t,G_t,\hat{A}_t)\right\|_2^2$$

- 总损失为三者直接相加：$\mathcal{L}_{\text{total}}=\mathcal{L}_{2D}+\mathcal{L}_{3D}+\mathcal{L}_{\text{ACT}}$。用大白话说：一个模型同时学"画面接下来会怎样""3D 几何接下来会怎样""看到 3D 几何变化该怎么动"，三件事在一次前向里一起对齐，而不是先训练一个视频/几何预测器再单独接一个动作头。

训练分预训练+后训练两阶段，且与以往工作（通常只在其中一个阶段做联合建模）不同，WSA 在两阶段都共同学习三个目标。

### 2.5 预训练数据配方

总计 6000 小时示范数据，覆盖 8 种机器人本体、300+ 控制任务、50+ 原子技能、692M 帧：

| 数据源 | 类型 | 帧数 | 任务数 | 采样权重 | 参与损失 |
|---|---|---|---|---|---|
| InternData-A1 | 仿真 | 396M | 70 | 0.47 | $\mathcal{L}_{2D}$、$\mathcal{L}_{3D}$、$\mathcal{L}_{\text{ACT}}$ |
| RoboTwin | 仿真 | 17M | 50 | 0.07 | $\mathcal{L}_{2D}$、$\mathcal{L}_{3D}$、$\mathcal{L}_{\text{ACT}}$ |
| AgiBot-World | 真机 | 206M | 217 | 0.17 | $\mathcal{L}_{2D}$、$\mathcal{L}_{3D}$、$\mathcal{L}_{\text{ACT}}$ |
| RoboChallenge | 真机 | 5M | 30 | 0.19 | $\mathcal{L}_{2D}$、$\mathcal{L}_{3D}$、$\mathcal{L}_{\text{ACT}}$ |
| EgoDex | 人类第一视角 | 68M | 194 | 0.10 | $\mathcal{L}_{2D}$、$\mathcal{L}_{3D}$（无动作监督）|

真机数据仅约 1000 小时（对应文中真机~1000h/仿真~5000h/人类第一视角~900h 的构成），远低于同类 RFM 通常依赖的数万小时真机数据规模。

两个模型规模：WSA₁-B（3B 参数，基于 Qwen3-VL-2B 初始化）与 WSA₁-L（6B 参数，基于 Wan2.2-5B 初始化）。

## 三、实验结果

### 3.1 真机实验

两种机器人：单臂 AgileX PiPER（7-DoF，3 相机）、双臂 ARX Lift2（14-DoF，3 相机）；7 项桌面操作任务（RGB 方块排序、物品整理、垃圾清理、笔筒摆放、玩具箱整理、扫垃圾、拉链笔袋开合），每任务 30 组随机初始配置评测，指标为成功率 SR 和完成度 C（子任务完成比例）。

| 方法 | 类型/规模 | 平均 SR (%) | 平均 C (%) |
|---|---|---|---|
| π0 | VLA 3B | 33.8 | 43.2 |
| π0.5 | VLA 3B | 54.9 | 63.3 |
| InternVLA-A1 | WAM 3B | 39.2 | 51.3 |
| **WSA₁-B** | WSA 3B | 77.5 | 82.7 |
| **WSA₁-L** | WSA 6B | 80.3 | 86.5 |

WSA₁ 在全部 7 个任务上都取得单任务最高 SR（如 Toy Box Organization：WSA₁-B 60% vs π0.5 13%；Sweep Trash：WSA₁-L 85% vs π0.5 26%），单臂/双臂两种本体上表现一致。

### 3.2 仿真基准

**RoboTwin2.0（hard 设置，50 任务平均 SR）：**

| 方法 | 开源状态 | 规模 | 类型 | SR (%) |
|---|---|---|---|---|
| Qwen-VLA | 闭源 | 5B | VLA | 87.2 |
| Being-H0.7 | 闭源 | 3B | WAM | 89.6 |
| MotuBrain | 闭源 | - | WAM | 96.1 |
| π0 | 开源 | 3B | VLA | 58.4 |
| π0.5 | 开源 | 3B | VLA | 76.8 |
| Motus | 开源 | 8B | WAM | 87.0 |
| InternVLA-A1 | 开源 | 3B | WAM | 89.6 |
| LingBot-VA | 开源 | 7B | WAM | 91.5 |
| Fast-WAM | 开源 | 6B | WAM | 91.8 |
| **WSA₁-B** | 开源 | 3B | WSA | 92.7 |
| **WSA₁-L** | 开源 | 6B | WSA | **93.1** |

WSA₁-B 以仅 3B 参数超过参数更大的 Motus（8B）、LingBot-VA（7B）等开源 WAM，为开源模型中最佳，但仍不及闭源的 MotuBrain（96.1%）。

**LIBERO（四个子集平均 SR）：** WSA₁-B 97.4%，WSA₁-L 98.2%，与当前最优 Lingbot-VA（98.5%）接近但未超过；明显优于 π0（94.4%）、SpatialVLA（78.1%）等。

### 3.3 消融实验

**WSA 建模组件消融**（RoboTwin2.0，基线为 Qwen3-VL-2B + 随机初始化扩散动作头的 π-style VLA）：

| 配置 | 平均 SR (%) |
|---|---|
| 仅动作（Baseline） | 60 |
| + 视觉思考（2D） | 76 |
| + 3D 世界预测 | 75 |
| 完整 WSA 建模 | 80 |

视觉思考对依赖视觉语义/外观变化的任务（如捡瓶子、开微波炉）帮助更大；3D 世界预测对需要精细空间几何理解的接触密集型任务（如转 QR 码、堆叠积木）帮助更大，二者互补。

**预训练策略消融**（RoboTwin2.0）：

| 配置 | 平均 SR (%) |
|---|---|
| 从零训练（Baseline） | 60 |
| WSA 仅后训练阶段应用（无预训练） | 80 |
| WSA 仅预训练阶段应用（后训练只优化动作损失） | 89 |
| WSA 预训练+后训练均应用（完整方案） | 93 |

预训练阶段引入 WSA 联合建模带来最大单项增益（80%→89%），预训练与后训练均保持三目标联合优化可进一步提升到 93%。

### 3.4 定性分析

论文展示了"blocks ranking RGB"和"stack the bowl on the plate"两个任务的四联可视化：当前观测+指令 → 规划的子目标图像 → 预测的目标 3D 世界（经 Depth-Anything 渲染为点云和深度图）→ 实际执行的 3D 动作轨迹。结果显示预测视觉计划能较好捕捉任务目标并与生成的 3D 世界对齐，动作轨迹与预测的 3D 目标状态高度一致。

## 四、局限性

论文正文未设独立的 Limitations 小节，以下为通读全文后可识别的客观局限：

- **未报告推理效率/延迟**：WSA₁ 推理时需同时跑通 2D 子目标预测、3D 场景预测和扩散式动作生成三个专家，相比单一动作头的 VLA（如 π0）理论上会引入更高的计算开销和延迟，但全文未给出任何 FPS/推理时延/显存对比数据；
- **仍落后于闭源 SOTA**：在 RoboTwin2.0 上 WSA₁-L（93.1%）仍明显低于闭源模型 MotuBrain（96.1%），"开源最优"的结论成立，但距离当前上限仍有约 3 个百分点差距；
- **评测任务集中于短时程桌面操作**：真机 7 个任务和仿真基准均为桌面级抓取/放置/整理类短程操作，未覆盖长时程移动操作、可变形物体或高精度装配等更具挑战性的场景；
- **对上游预训练模型的强依赖**：2D-SE 初始化依赖 Qwen3-VL/Wan2.2，3D 监督信号依赖 Depth-Anything，视觉 token 目标依赖 COSMOS tokenizer，模型性能与可复现性都与这些外部基础模型的质量和可获得性强耦合；
- **消融粒度有限**：数据配方消融只给出"预训练/无预训练"和"WSA 各分量开关"的整体对比，未单独消融每个数据源（如仅 EgoDex 或仅 AgiBot-World）对最终性能的边际贡献，无法判断哪类数据源对 3D 世界建模贡献最大。

## 五、评价与展望

**优点**：WSA 较早明确提出"3D 世界预测与动作生成应双向互相约束"而非单向"预测后执行"，用清晰的条件分布记号把 VLA、2D WAM、3D WAM（如 GeoPredict）与自己的方法放在统一框架下比较，思路表达清楚。实验证据也比较扎实：同时在真机（两种本体）、RoboTwin2.0（clean/hard 双设置）、LIBERO 三套基准上验证，并给出了针对性的组件消融与训练策略消融，支持了"3D 世界-动作双向约束+预训练阶段联合学习"这一核心主张，而不只是堆参数量刷分——WSA₁-B 仅 3B 参数即超过 8B 的 Motus 和 7B 的 LingBot-VA。真机仅用约 1000 小时数据即取得远超 π0.5 的成绩，对降低具身数据采集门槛有实际参考价值。

**与已有公开工作的关系**：相较于 2D WAM（如 Fast-WAM 提出"操作泛化性提升来自协同学习而非显式视频预测本身"的观点），WSA 把这一协同学习显式提升到 3D 空间；相较于此前的 3D WAM（如 GeoPredict、PointWorld），WSA 的核心区别是把"预测然后执行"的单向流水线改造成 3D 场景与动作互为条件的双向依赖，这是本文相对最接近工作的主要增量。

**开放问题与可能的改进方向**：(1) 论文未讨论多专家结构带来的推理时延，这对实时闭环控制（尤其高频伺服任务）是否可行仍待验证；(2) 双向注意力耦合 3D 场景预测与动作生成，理论上存在预测误差累积并反噬动作生成的风险，但论文未报告长 horizon rollout 下的误差累积分析；(3) 当前仅验证了桌面级操作，3D 世界建模范式在移动操作、可变形物体或多物体复杂遮挡场景下的有效性尚待检验；(4) 消融未拆解各数据源（尤其占比达 0.47 的 InternData-A1 仿真数据）的边际贡献，后续工作可进一步研究仿真数据比例与 3D 世界建模精度之间的关系，以及是否可以进一步压缩真机数据需求。
