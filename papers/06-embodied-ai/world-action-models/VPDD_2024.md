## 一句话总结

VPDD 用统一的 VQ-VAE 把人类视频和机器人视频压缩进同一套离散视频 token 空间，再用一个统一的离散扩散模型先在大规模无动作 Ego4D 人类视频上做"预测未来视频"的自监督预训练，然后冻结视频去噪网络、仅用少量机器人示范微调一个动作去噪网络（以预测视频的中间隐藏表征作为"前瞻"信号指导动作生成），在 Meta-World（50 任务）和 RLBench（16 任务）上用远少于同类方法所需的机器人数据取得 SOTA 成功率，并在相机视角/桌面纹理偏移的分布外场景下展现出明显更强的泛化能力。

## 一、问题与动机

训练通用具身智能体依赖大量带动作标注的机器人数据，而这类数据的采集（遥操作、运动学求解器）成本高、耗时长，且不同机器人本体之间存在显著差异。相对地，互联网上存在海量无动作标注的人类交互视频（如 Ego4D），蕴含丰富的物体、场景、手物交互语义，但如何把这些知识迁移到机器人策略学习上仍面临三个核心挑战：(i) 人类与机器人之间的视觉域差（domain gap）阻碍知识迁移；(ii) 人类视频中的行为模式复杂、多样且噪声大，难以提取有用结构；(iii) 视频、动作、语言等多模态数据需要一个可扩展且富有表达力的统一模型架构处理。此前方法多聚焦于从人类视频学习视觉表征、轨迹表征、奖励函数或世界模型，但大多只捕捉逐步（step-by-step）转移关系，未能建立面向长程行为的动态理解。

## 二、核心方法

VPDD（**V**ideo-based **P**olicy learning via **D**iscrete **D**iffusion）分为视频 tokenizing、大规模无动作视频预训练、少样本机器人微调三个阶段。

**1. 统一视频 tokenizing。** 借助 VQ-VAE（沿用 VideoGPT 的架构）把人类和机器人视频编码进同一离散 latent 空间：3D 卷积编码器在时空三个维度各下采样 4 倍（共 64 倍压缩），随后接 6 个注意力残差块；codebook 含 2048 个 256 维 code。编解码器与 codebook 端到端联合训练：

$$\mathcal{L} = \|x-\hat x\|_1 + \|\mathrm{sg}[E(x)]-z_q\|_2^2+\beta\|\mathrm{sg}[z_q]-E(x)\|_2^2$$

用大白话说：先把连续视频压成一串离散"视觉词元"，再用这串词元反解回视频，训练目标既要重建准确，也要让编码器输出和 codebook 里最近的词向量相互靠拢，这样人类视频和机器人视频就能被投影到同一张"词表"上，天然抹平了两者在像素层面的域差。

**2. 离散扩散（mask-and-replace，吸收态）。** 在 codebook 之外引入一个 [MASK] token，前向扩散按转移矩阵 $Q_k$ 逐步把干净 token 替换为其他 token 或 [MASK]：

$$q(x_k\mid x_{k-1})=\mathrm{Cat}(x_k;p=Q_k x_{k-1})$$

其中每个 token 以概率 $\alpha_k$ 保持不变、以概率 $\beta_k$ 被随机替换为其他普通 token、以概率 $\gamma_k=1-\alpha_k-J\beta_k$ 被替换为 [MASK]。反向过程用 $x_0$-参数化（网络直接预测干净数据分布而非逐步噪声）以提升训练稳定性并支持跳步加速推理。

**3. 统一转移矩阵，联合视频与动作。** 把视频 token（codebook 大小 $J$）和离散化动作 token（类别数 $J^*$）的转移矩阵拼成一个 $(J+J^*+1)\times(J+J^*+1)$ 的分块矩阵，用各自的 $\beta_k,\beta_k^*$ 控制两种模态被替换的概率，使视频预测和动作生成可以纳入同一套离散扩散过程建模，为预训练与微调之间提供了一致的接口。

**4. 两阶段训练。** 预训练阶段仅用无动作视频，最大化未来视频 token 的条件对数似然，动作侧被整体 mask 掉；去噪网络 $p_{\theta_1}$ 用 Perceiver IO Transformer（latent cross-attention 处理超长视频 token 序列，避免标准 Transformer 的 $O(n^2)$ 复杂度），条件包括 CLIP 编码的语言指令、历史视频 token 和扩散时间步，优化变分下界 $\mathcal{L}_{vlb}=\mathcal{L}_0+\sum_{k=2}^K \mathcal{L}_{k-1}+\mathcal{L}_K$。微调阶段冻结 $p_{\theta_1}$，用 GPT2 Transformer 作动作去噪网络 $p_{\theta_2}$，除历史/语言/时间步外，额外接入 $p_{\theta_1}$ 每个去噪步产出的隐藏表征 $z_{\tilde x_0^v}=\mathrm{Softmax}(\mathrm{MLP}(z_{\tilde x_0^v}))$ 作为对未来视频的"前瞻（foresight）"信号，据此逐步去噪出动作 token。用大白话说：预训练阶段先学会"看到过去、脑补未来会发生什么"，微调阶段只需在这份脑补的基础上学"该怎么做才能让脑补的未来发生"，且脑补信息会随扩散步数从粗到细持续精化，而不是像某些表征学习基线那样全程只给一份固定不变的表征。

**动作离散化与数据。** Meta-World 动作为末端执行器 3D 位移 + 归一化夹爪力矩，所有连续维度统一离散化为 48 个 bin（4 个整数序号表示）；RLBench 动作为夹爪开合二值 + 6-DoF 位姿，位置离散化为 360 个 bin，旋转离散化为绕 3 轴各 1 度精度的欧拉角 bin。预训练人类视频来自 Ego4D，过滤出含人-物交互片段并按 8 帧间隔切分，共 996,177 个片段（约 400 万帧）；机器人视频方面 Meta-World 用规则脚本策略采集 20 条示范/任务，RLBench 用运动规划器采集 10 条示范/任务（16 个任务，4 视角 RGB）。超参数：动作预测 horizon $H=4$，未来视频片段数 $M=1$，历史长度 $h=20$，扩散时间步 $K=100$，预训练 batch size 20 / 微调 40，Adam 学习率 $2\times10^{-4}$，训练 $2\times10^6$ 步，单卡 RTX 3090 完成全部实验。

## 三、关键结果

**Meta-World（MT50-rand，50 个随机目标任务，20 demo/任务，3 个随机种子，50 episode/任务评测）：**

| 方法 | 平均成功率 | 说明 |
|---|---|---|
| VPDD（Ours） | **56.57 ± 0.41%** | 人类+机器人视频预训练 + 少样本微调 |
| VPDD-w/o-human | 相对 VPDD 低 6.9%（相对值） | 消融：预训练阶段去掉人类视频 |
| R3M-Diffusion | 相对 VPDD 低 16.4% | R3M 视觉表征 + 同架构离散扩散 |
| VC-1-Diffusion | 相对 VPDD 低 26.5% | VC-1（ViT-L, MAE）视觉表征 |
| MTDIFF-P | 比 R3M-Diffusion 低 26.0% | 连续扩散 + Transformer 的多任务 RL SOTA |
| Video-MTDT | 比 R3M-Diffusion 低 60.6% | Decision Transformer 的视频 token 版本 |

**RLBench（16 个任务，10 demo/任务，25 episode/任务，3 seed；VPDD 仅用 RGB，RVT/PERACT 额外使用深度图）：** VPDD 相对 PERACT 和 RVT 平均成功率提升 **1.24 倍**；在 *put in cupboard*、*insert peg*、*stack cups*、*place cups* 等对 3D 场景理解要求较高的任务上，专门设计的体素/多视角 3D 表征方法 RVT、PERACT 均表现挣扎，而 VPDD 取得明显更高的成功率。

**视频预测质量（FVD，越低越好，Meta-World 上 32 个采样视频）：** UniPi 264.66 vs VPDD（Ours）**235.81**，说明预训练得到的视频预测模型能生成动态一致、可用于规划的未来视频。

**分布外泛化（相机位置/桌面纹理偏移，button-press-v2 与 handle-press-v2 任务）：** 相机位置偏移场景下，VPDD 相对 VPDD-w/o-human 成功率高 63%，相对 R3M 高 252%，证明大规模人类视频预训练确实注入了可跨场景迁移的常识性动态知识。

**跨任务泛化（留出 3 个未见 Meta-World 任务：预训练阶段仅用其余 47 个任务，微调阶段用 0.2M 步在留出任务上适配，与用全部 50 任务训练的 oracle 对比）：**

| 未见任务 | VPDD | VPDD（oracle） |
|---|---|---|
| hand-insert-v2 | 32% | 36% |
| bin-picking-v2 | 78% | 84% |
| door-unlock-v2 | 100% | 100% |

差距很小，显示出一定的跨任务泛化潜力。

**样本效率消融：** 微调阶段仅用 1 条机器人示范也能维持接近的成功率；5 条以上示范起，成功率随示范数量近似线性增长。预训练阶段人类视频数量越多，下游成功率（尤其是分布外泛化设置下）提升越明显。

**与 SODA（另一扩散式表征学习基线，同一视频预测目标训练 encoder）对比：** VPDD 优于 SODA，作者将其归因于 VPDD 在动作去噪的每一步都提供从粗到细逐步精化的视频表征，而 SODA 在所有去噪步上使用恒定不变的表征。

## 四、评价与展望

**优点。** VPDD 用同一套离散扩散框架同时覆盖"无动作视频的自监督预测"和"有动作视频的策略学习"两个任务，预训练-微调之间的接口干净统一，不像 UniPi 等工作那样需要额外训练任务特定的逆动力学模型来把预测视频转成可执行动作。实验证明仅用 5~20 条/任务的机器人示范加人类视频预训练，即可匹配甚至超过依赖更多机器人数据、专门 3D 表征（点云/多视角体素，如 RVT、PERACT）且额外使用深度图输入的方法，而 VPDD 只用 RGB。分布外场景（相机、纹理、任务）上的泛化结果也支持了"大规模无动作人类视频预训练能提升下游鲁棒性"这一假设。

**局限与开放问题。** 论文自陈生成视频存在模糊、跨视角一致性不足的问题（附录展示了失败样例：机械爪姿态在不同视角预测中出现偏差），作者也承认其主要贡献不在视频生成质量本身。所有实验均在仿真基准（Meta-World、RLBench）上完成，未见真实机器人部署结果，人类视频到机器人动作的迁移效果在真实场景中的稳健性尚待验证。离散化的动作空间（48/360 个 bin）在面对更高精度连续控制任务时的可扩展性也未知。与同期工作 GR-1（同样用人类视频预训练类 GPT 架构预测未来观测）相比，VPDD 用离散扩散取代自回归、预测目标从单帧图像换成视频 token 序列，但两者均缺乏真实机器人上的大规模验证，也未直接互相比较。相较 UniPi 采用连续扩散 + 超分辨率提升时序一致性的路线，VPDD 完全依赖 VQ-VAE 视频 token 自身的时序一致性，这可能是其生成视频细节偏模糊的部分原因。作者提出的后续方向包括利用 Ego-Exo4D 等第一/第三人称配对数据增强跨视角一致性，以及扩大训练数据规模与模型容量。
