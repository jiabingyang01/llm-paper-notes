## 一句话总结

CoMo 用一个**去掉向量量化（VQ-free）的连续潜在运动 IDM-FDM 框架**，配合**早期时间差分（Td）** 与**时间对比学习（Tcl）** 两个机制来抑制"捷径学习"，从无动作标注的互联网视频中抽取连续潜在运动作为伪动作标签，与真实机器人数据统一联合训练策略；在 LIBERO 上把成功率从 70.4% 提升到 **80.1%**，在 CALVIN ABC→D 的自回归策略上把平均完成任务数从 1.878 提升到 **3.070**（扩维后 3.247）。

## 一、问题与动机

大规模机器人真实数据稀缺、多样性低、异构性高,难以支撑可扩展的策略学习。一条流行范式是从海量无动作标注视频中学习**潜在运动（latent motion）**,再据此为视频生成伪动作标签,与真机数据一起训练统一策略。代表工作（LAPA、UniVLA、Moto 等)采用 **inverse dynamics encoder + forward dynamics decoder** 架构,配合 VQ-VAE 目标,把帧对之间的运动量化成离散 token。

这类方法必须用**很小的码本**(LAPA 码本 8、UniVLA 码本 16)来抑制"**捷径学习**（shortcut learning）"。所谓捷径学习指:inverse dynamics encoder 为了让 decoder 重建未来帧,会图省事地把未来帧的静态背景信息直接编码进"运动"表征里,而不是聚焦真正的前景运动——因为像素级细节最容易被重建。这样学到的表征退化成一个"未来帧预测器",作为动作预测机制毫无用处。小码本虽能缓解此问题,却带来两个新的痛点:

- **信息损失**:真实运动本质上是连续且细粒度的,离散量化不可避免丢失细节,泛化到新运动模式的能力受限。
- **分布错配**:离散潜在运动与连续机器人动作之间存在天然的分布 gap,阻碍二者在统一策略里的联合学习。

作者提出的核心问题是:**能否直接从无动作视频里学习连续潜在运动?** 连续表征能更精确刻画帧间细粒度变化,并与连续机器人动作分布天然一致。但直接去掉 VQ 会导致严重的捷径学习——模型直接坍缩为把静态未来帧背景当作连续运动。CoMo 就是要在**不依赖量化瓶颈** 的前提下解决这一坍缩。

## 二、核心方法

CoMo 沿用 IDM(inverse dynamics model,反向动力学)+ FDM(forward dynamics model,前向动力学)范式,但**完全去掉向量量化**,并引入两个协同机制。

### 2.1 早期时间差分（Td）

给定当前帧 $O_t$ 与未来帧 $O_{t+n}$,用一个共享的 MAE 预训练 ViT-L 分别抽取 token 级特征 $F_t$ 与 $F_{t+n}$。逐元素相减得到 token 级特征差分:

$$D_t = F_{t+n} - F_t$$

关键操作是:在送入编码器前**显式删除未来帧特征 $F_{t+n}$**,只把当前帧特征 $F_t$ 与差分 $D_t$ 拼接成组合表征 $[F_t, D_t]$。随后按 Moto 的做法拼上一组可学习的 query embedding(Motion Query),在标准多层 Transformer(Motion Q-former)里做全注意力交互,取 query 的输出作为运动表征 $Z_{(t,t+n)}$。

> **用大白话说**:直接把未来帧喂给编码器,它就会"抄答案"——把未来背景直接搬过来当运动。Td 把未来帧藏起来,只留下"当前长啥样 + 帧间变了什么(差分)",既放大了运动信号、又抬高了走捷径的难度。但差分是稀疏的,当运动维度较大时不可避免会混入无关信息。

### 2.2 时间对比学习（Tcl）

为了压制差分里残留的无关信息,CoMo 再叠加一个对比学习。在 $O_{t+n}$ 附近采样 $O_{t+n+\delta}$($\delta \in [-n/5, n/5]$),用同一 IDM 得到若干运动表征,用 InfoNCE 拉近/推远:

$$S_1 = S(Z_{(t,t+n+\delta)}, Z_{(t,t+n)}),\quad S_2 = S(Z_{(t,t+n+\delta)}, Z_{(t+n,t)}),\quad S_3 = S(Z_{(t,t+n)}, Z_{(t+n,t)})$$

$$\mathcal{L}_{tcl} = -\log \frac{e^{S_1}}{e^{S_1} + e^{S_2} + e^{S_3}}$$

其中 $S$ 是余弦相似度;$Z_{(a,b)}$ 表示从有序帧对 $(O_a, O_b)$ 抽取的运动(即 $a \to b$ 方向)。正样本对 $(Z_{(t,t+n+\delta)}, Z_{(t,t+n)})$ 是两个**同向、只差一点时间偏移** 的运动;负样本对则是**时间方向被翻转**($Z_{(t+n,t)}$,即 $t+n \to t$)的运动。

> **用大白话说**:同一段运动即使起止帧稍微挪一点,方向和"怎么动"应该几乎一样,所以拉近;而把时间倒放的运动方向相反,应该推远。这逼表征去编码"how"(运动本身)而不是"what/where"(未来帧里有什么物体、在哪)。

作者强调 Td 与 Tcl 这两个机制**缺一不可、协同互补**:单靠 Td,差分仍会混入无关背景;单靠 Tcl,对比又只学到未来帧前景物体的身份("what/where"),忽略时间运动模式("how")。二者合起来,才能保证连续潜在运动**既聚焦有意义前景、又富含动作相关的运动线索**,且**不再需要小码本或低维瓶颈这类强约束,因而更可扩展**。

### 2.3 FDM 与整体训练

FDM 以 $Z_{(t,t+n)}$ 和当前帧 $O_t$ 为条件重建未来帧:先对 $O_t$ 做线性 patch embedding,再把池化后的运动特征加到 patch embedding 上得 $E(O_t, Z_{(t,t+n)})$,过若干 Transformer 块,再用卷积 + pixel shuffle 重建 $\hat{O}_{t+n}$。整体损失 = 加权 InfoNCE + 重建损失 + 感知损失。潜在运动嵌入维度默认仅 16,motion query 数 8,IDM 4 层 / FDM 12 层。

### 2.4 统一策略联合学习

对无动作视频集 $\mathcal{D}_V$,用训练好的 CoMo IDM 抽取连续潜在运动 $z_t$,把轨迹增广成 $[(o_0, z_0), \ldots, (o_T, z_T)]$;真机数据集 $\mathcal{D}_R$ 则是 $[(o_0, a_0), \ldots]$。由于动作 $a$ 与潜在运动 $z$ **都是连续分布**,可在 $\mathcal{D}_R \cup \mathcal{D}_V$ 上用统一生成式策略做联合模仿学习,仅需给两个数据源各配一个轻量 head,无需复杂的多阶段预训练/微调或"先运动后动作"两阶段流程。作者同时实现了 diffusion 策略(DiT/RDT-1B block,DPM-Solver++ 5 步)和自回归策略(GPT 式,动作 chunk 5 / 运动 chunk 2)。

### 2.5 两个免策略评估指标

为在不训练策略的前提下诊断潜在运动质量,作者提出两个互补指标:

- **动作预测 MSE**:训一个 MLP 从离线抽取的 $z_t$ 回归真值动作 $a_t$,报告 MSE,衡量表征对**动作相关信息** 的编码能力(越低越好)。
- **过去到当前 / 未来到当前相似度 S-PCFC**:在运动为主、背景变化极小的机器人数据上,度量 $z(O_{t-n},O_t)$ 与 $z(O_{t+n},O_t)$ 的余弦相似度:

$$\text{S-PCFC}(t) = \frac{z(O_{t-n},O_t)^\top\, z(O_{t+n},O_t)}{\lVert z(O_{t-n},O_t)\rVert_2\, \lVert z(O_{t+n},O_t)\rVert_2}$$

> **用大白话说**:"从过去走到现在"和"从未来走到现在"是两段方向相反的运动,理应不相似。如果表征里塞满了静态背景噪声,这两段就会因共享背景而"看起来很像",S-PCFC 就偏高。所以 **S-PCFC 越低,说明背景冗余越少、运动保真度越好**。经验上 MSE 和 S-PCFC 两者**同时较低** 时,下游策略成功率最高。

## 三、实验结果

**数据与设置**:潜在运动 IDM 在**纯域外** 互联网视频上训练——SAM-V、EgoVid、Droid 三个数据集各 40,000 段,共 120,000 段,以此检验零样本跨域迁移。LIBERO 联合训练时每个任务**仅用 10 条真机轨迹**(远低于常规 50 条),其余靠伪标签;CALVIN 用 ABC→D 最难设置。

**表 1:LIBERO 消融（成功率↑ / MSE↓ / S-PCFC↓,均值行)**

| 变体 | 成功率↑ | MSE↓ | S-PCFC↓ |
|---|---|---|---|
| Future features(未来帧特征) | 73.8 | 2.1442 | 1.0000 |
| Dis.(离散量化) | 75.9 | 5.6743 | **-0.0678** |
| Con.(朴素连续) | 75.2 | 1.6256 | 0.9268 |
| Con.+Td | 76.9 | 1.5209 | 0.8887 |
| Con.+Tcl | 77.9 | 1.3054 | 0.6229 |
| **Con.+Td+Tcl（CoMo)** | **80.1** | **1.2588** | 0.5496 |

关键读数:CoMo 在成功率、MSE 上均最优,比 Dis. 高 **4.2** 个点(75.9→80.1);Dis. 虽然 S-PCFC 最低(-0.0678),但成功率并不占优——说明**单看 MSE 或单看 S-PCFC 都不足以保证策略性能,二者同时低才对应最高成功率**。分套件上 CoMo 达 Spatial 80.3 / Object 97.0 / Goal 81.0 / Long 62.0。

## 四、局限性

- **协同增益在部分任务上仍偏小**:作者自承 CoMo 本质上更利好细粒度任务,常规基准里某些任务的 co-training 提升有限,需要更大规模、更复杂的真机基准来充分体现优势。
- **单帧对、单视角建模**:当前 IDM 只用一对帧、单一视角,面对遮挡时运动抽取精度受限;作者把扩展到多帧、多视角建模列为重要方向。
- **依赖两个代理指标**:MSE / S-PCFC 是免策略诊断,但论文也指出二者单独都不足以预测策略性能,需组合使用;其与真实成功率的相关性偏经验性,缺乏理论保证。
- **对比学习的负样本设计较单一**:负样本主要来自"时间翻转",尚未探索更丰富的难负样本;$\delta$ 的取值范围 $[-n/5, n/5]$ 属人工设定。

## 五、评价与展望

**优点。** 这篇工作把"latent action / latent motion 预训练"这一路线里长期靠**小码本 VQ 强行压制捷径学习**的隐性妥协挑明,并给出一条干净的连续化替代方案。两个机制的动机都很物理:Td 用"藏未来帧 + 特征差分"抬高抄答案成本,Tcl 用时间翻转作负样本逼表征学"运动方向"而非"物体身份",且实验(Fig 3、表 1)清楚地展示了二者缺一不可与可扩展性。更实际的贡献是**连续潜在运动与连续机器人动作分布对齐**,从而把 co-training 简化成"共享主干 + 双 head",省去 LAPA/Moto 式的两阶段(先运动后动作)流程,这对工程落地很友好。提出的 S-PCFC 也是一个便宜、可复现的表征诊断工具,有一定通用价值。

**与其他公开工作的关系。** CoMo 直接对标 LAPA、UniVLA、Moto/Moto-GPT 这一系的**离散潜在动作**;相较 GR00T(预量化潜在运动)、Dynamo(协方差正则)、ATM(2D 点轨迹)、GR2(未来帧特征)等不同预测信号,它在 LIBERO 上给出了统一的对比并领先。它与生成式世界模型路线(如 AdaWorld、DreamVLA、Cosmos policy 等)在"用视频监督策略"这一目标上一致,但走的是"抽取显式运动 token 作伪标签"而非"预测未来帧/生成数据"的路子,推理与训练成本更低。

**开放问题与可能改进。** (1)**理论层面**:S-PCFC/MSE 与成功率的关系仍是经验相关,若能给出"何种潜在运动可作为最优伪动作"的信息论刻画,会更有说服力。(2)**多帧多视角**:作者已指出遮挡下单帧对失效,引入时序上下文或多视图几何(乃至深度)有望进一步提纯运动。(3)**跨本体动作空间**:表 4 已初步验证 14 维双臂与 128 维人形,但绝对/相对动作空间、异构自由度之间的潜在运动可迁移性还值得系统研究。(4)**负样本与增广**:时间翻转之外,速度/尺度扰动、跨视频难负样本可能进一步提升"how"的判别性。(5)**规模化验证**:目前基准以 LIBERO/CALVIN + 5 个真机任务为主,数据与本体规模仍有限,连续潜在运动的 scaling law 尚待更大规模检验。

总体看,CoMo 是 latent-action 预训练从"离散/小码本"向"连续/可扩展"迁移过程中一篇动机清晰、消融扎实的工作,其"Td+Tcl 抑制捷径学习"的思路与 S-PCFC 诊断指标对后续从无动作视频挖运动伪标签的研究有直接参考价值。

## 参考

1. Moto: Latent Motion Token as the Bridging Language for Robot Manipulation（Chen et al., ICCV 2025）——CoMo 直接沿用其 IDM-FDM + Motion Q-former 架构,主要对标的离散基线。
2. LAPA: Latent Action Pretraining from Videos（Ye et al., ICLR 2025)——小码本(8)离散潜在动作的代表,CoMo 挑战其量化瓶颈。
3. UniVLA: Learning to Act Anywhere with Task-centric Latent Actions（Bu et al., 2025)——码本 16 的离散潜在动作路线,CoMo 的连续化对照。
4. CLAM: Continuous Latent Action Models for Robot Learning from Unlabeled Demonstrations（Liang et al., 2025)——同样探索连续潜在动作,与 CoMo 思路相关的并行工作。
5. TDN: Temporal Difference Networks for Efficient Action Recognition（Wang et al., CVPR 2021)——CoMo 早期时间差分机制的灵感来源。
